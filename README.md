A demo project about using PHPUnit.
This project implement a simple calculator by php

Requirements
---

* php >=5.6

Installation
---

```
$ git clone git@github.com:oanhnn/demo-phpunit-calculator.git /path/to/project
$ cd /path/to/project
$ composer install
$ ./vendor/bin/phpunit
```
